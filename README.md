# E-commerce Full Stack

Aplicación web full stack construida con **Nest.js** (backend) y **React** (frontend).

## Stack

- **Backend**: Nest.js + TypeORM + PostgreSQL
- **Frontend**: React + TypeScript + Vite
- **Auth**: JWT

## Estructura

```
api/      # Backend Nest.js
client/  # Frontend React
```

## Getting Started

### API

```bash
cd api
npm install
npm run start:dev
```

### Client

```bash
cd client
npm install
npm run dev
```
